Task: Add DELETE functionality

  • Base the lab on the code in the "demo-04-front-controller-flow.zip" project (see Course Modules overview).
  • Add functionality to delete an Item from the Shopping list.
  • Hints:
    • You must update the following files:
      • ItemDAO.java
      • ItemDAOMemorySingleton.java
      • FlowManager.java
      • ActionMapper.java
      • shoppinglist.jsp
    • You must create a new Action class (for the delete operation).
    • Extra: Add a verification page before actually deleting Item.
      (Note! Reuse the ViewItemAction class for the uri (mapping) for the verification).
